# page.hex

### Develop

```
npm install hexo-cli -g
npm install hexo-render-pug hexo-renderer-stylus --save
git clone -b page.hex git@github.com:aylax/aylax.github.io.git /path/to/you
git submodule init
git submodule update
npm install
hexo server
```
